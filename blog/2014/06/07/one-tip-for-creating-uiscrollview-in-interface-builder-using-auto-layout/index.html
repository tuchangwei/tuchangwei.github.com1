One Tip for Creating UIScrollView in Interface Builder Using Auto Layout

When I started using Auto Layout, I was confused about it, what is the meaning of the Constraints, how to organize them to ensure the frame of a view is appropriate. I looked up some articles to help me understand it until I found an author said “you just do it, after youd do some, you will understand how it works”.

Thanks for his advice. Now, after much practise, I think I can use it well.

About the UIScrollView, I want to say it is special when you use Auto Layout to resize it.

When you use Auto Layout,its setContentSize method is unavailable, somebody says we can invoke the method in viewDidAppear method, then we can resize its content size, I don’t think it is a good idea.

The right way is

  1. Drag a view as the content view of the scroll view.
  2. Set the constraints of the content view, if the height is higher than the scroll view’s, then it can scroll auto.
